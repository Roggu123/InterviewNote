# InterviewNote
题目描述

计算字符串最后一个单词的长度，单词以空格隔开。

<br>输入描述:<br>
<br>一行字符串，非空，长度小于5000。<br>

<br>输出描述:<br>
<br>整数N，最后一个单词的长度。<br>

## 示例1
<br>输入<br>
<br>hello world<br>
<br>输出<br>
<br>5<br>
## 思路
<br>1.单词位置，两个空格或一个空格与一个字母之间，设置一个标记区分前后空格<br>
<br>2.单词长度，非空格个数，设置一个变量存储数目<br>
<br>3.使用输入函数getline()<br>
<br>4.使用循环计数<br>
